Practica de Git

Primera version

U significa Untracked y quiere decir que ese archivo, no esta siendo gestionado por GIT

Segunda version

A significa added y quiere decir que, ese archivo, ha sido registrado para un futuro commit

Codigos:

Para configurar: git config --global user.name "Paulof89"

git config --global user.email "paulo.pef87@gmail.com"

git config --list

Para inciar: git Init Luego para agregar: git add .

git status: para saber la situacion actual

git commit "Commit inicial: index.hmtl y css/style.css"

git commit -m "Segundo commit: modificado inedx.html y agregado el reset css"

git log: muestra el historial y el hash

git log --oneline: version resumida del log

git chekaout: vuelve para atras

git chekaout "master": regresa para la ultima version
