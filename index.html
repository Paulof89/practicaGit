Practica de Git

Primera version

U significa Untracked y quiere decir que ese archivo, no esta siendo gestionado por GIT

Segunda version

A significa added y quiere decir que, ese archivo, ha sido registrado para un futuro commit

Codigos:

git Init: iniciar

git add . : agregar

git status: para saber la situacion actual

git commit "Commit inicial: index.hmtl y css/style.css"

git commit -m "Segundo commit: modificado inedx.html y agregado el reset css"

git log: historial y hash

git log --oneline: resumen del log

git checkout: switchea con otra rama

git branch: crea rama

git merge: une las diferentes ramas, desde el main
